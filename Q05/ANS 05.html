To change the content of the ‘Login / Register’ button to ‘Connect with us’ using the developer tool.

  • We have to visit the PW Skills website (https://pwskills.com/)
  • Then we have to right click anywhere on the web page and then select inspect
  • Then we have to find nav information and then find the information related to button
  • Then we have to to right click on button's information and select the Edit as HTML
  • then we have to cut the 'login/register' and then write 'Connect with us'
  • Then we successfully change the content of the ‘Login / Register’ button to ‘Connect with us’
